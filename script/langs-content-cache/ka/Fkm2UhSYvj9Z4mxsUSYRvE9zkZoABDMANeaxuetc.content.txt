კალიფორნიის სამზარეულო - კალიფორნიის შტატის კერძების და მათი მომზადების ტრადიციების ერთობლიობა. კალიფორნიის სამზარეულოზე ძლიერად აისახა შტატის ეთნიკური და კულტურული მრავალფეროვნება.   გავლენა იქონია ლათინური ამერიკის, ევროპის (კერძოდ ესპანეთის, იტალიის, პორტუგალიის) და აზიის (იაპონიის, ჩინეთის, ვიეტნამის, ტაილანდის) სამზარეულომ და ასევე შეერთებული შტატების სამზარეულოს დინებებმა და ტენდენციებმა მთლიანობაში. ასევე მისი ჩამოყალიბება განაპირობა ზღვასთან სიახლოვით გამოწვეულმა ზღვის პროდუქტების ხელმისაწვდომობამ, ჰოლივუდის ფილმების ინდუსტრიამ, განვითარებულმა მეღვინეობამ, თბილმა კლიმატმა და სოფლის მეურნეობის ტრადიციებმა. 


== ინგრედიენტები ==
მთავარ ინგრედიენტებში შედის ახალი ხილი, ბოსტნეული, ხორცი და ზღვის პროდუქტები (განსაკუთრებით გავრცელებულია კრაბი, ლობსტერი და კრევეტი), მოსახლეობაში ფასეულობს სწორი კვება. კალიფორნია აწარმოებს შეერთებული შტატების თხილის ზოგიერთი სახეობის, ნექტარინის, ზეთისხილის და კივის უმეტესობას და ერთ-ერთი წამყვანი შტატია ავოკადოს, ყურძნის, ლიმნის, მარწყვის, ატმის და ნესვის წარმოებაში. 


== სწრაფი კვება ==
სამხრეთი კალიფორნიის მოსახლეობის დამოკიდებულებამ ავტომობილებზე დიდ ქალაქებში გადაადგილებისთვის განაპირობა სწრაფი კვების drive-in და drive-thru რესტორნების პოპულარობა. აქ წარმოიშვა ასეთი რესტორანთა ქსელები, როგორც McDonald's [1], Taco Bell [2], Jack in the Box და სხვა. სწრაფი კვების ქსელების მენიუ განსხვავდება რეგიონის ეთნიკური შემადგენლობის მიხედვით, მაგ. მექსიკის საზღვართან პოპულარულია ტაკო, ბურიტო, ნაჩოსი და ა.შ. 


== ლათინური ამერიკის და ესპანეთის გავლენა ==
ისტორიულად კალიფორნია მექსიკის ნაწილი იყო, სადაც ცხოვრობდნენ მკვიდრი ამერიკელები, მექსიკელები და ესპანელი იმიგრანტები, ამიტომ მის სამზარეულოზე ესპანული და მექსიკული გავლენა ძალიან დიდია, განსაკუთრებით სამხრეთ კალიფორნიაში [3]. სწრაფი კვების ქსელებში მექსიკული კერძები ძალიან პოპულარულია. ტრადიციულად დამზადებული მექსიკური კერძები ასევე გავრცელებულია. მათ შორის ტორტილია, ტოსტადა, კესადილია, ენჩილადა, მენადო, პოზულე, მოლეს სოუზი, ჩილე რელენო, პუპუსერიასი, ტამალე და სხვა. 


== აზიის და ოკეანეთის გავლენა ==
კალიფორნიამ, როგორც ტერიტორიულად აზიასთან და ოკეანეთთან ერთ-ერთნა ყველაზე ახლოს მყოფმა შტატმა დიდი რაოდენობის აზიური მოსახლეობით აითვისა აზიური კერძების გარკვეული რაოდენობა. ამერიკაში სუშის პოპულარობა უდავოდ კალიფორნიაში დაიწყო, ტერმინი "კალიფორნიული როლი" აღნიშნავს სუშის, რომლის ძირითადი ინგრედიენტია ავოკადო. 


== კალიფორნიული პიცა ==
კალიფორნიული პიცა გამოირჩევა ტრადიციული პიცისთვის უჩვეულო ინგრედიენტების, მაგ. კვერცხის, იხვის სოსისის,  თხის ყველის გამოყენებით. მზადდება თხელი ცომით, როგორც იტალიური და ნიუ-იორკული პიცა და უმატდება კალიფორნიაში გავრცელებული ინგრედიენტები.  ასეთი პიცა სხვა შტატებშიც პოპულარულია. მის გამოგონების დროდ 1980-იანი წლები ითვლება, როდესაც ერთ-ერთი ცნობილი კალიფორნიელი პიცის მცხობელი ინგრედიენტების სხვადასხვა კომბინაციის შექმნას ცდილობდა.
