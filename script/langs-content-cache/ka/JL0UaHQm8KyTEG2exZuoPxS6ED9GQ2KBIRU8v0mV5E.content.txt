მეტალიკა — 1981 წელს დაარსებული ამერიკული გრემის მფლობელი მძიმე მეტალ ჯგუფი, რომლის წევრებიც არიან: ჯეიმზ ჰეტფილდი - წამყვანი ვოკალი და რიტმული გიტარა, ლარს ულრიხი - დრამი და დასარტყმელი საკრავები, კირკ ჰემეტი - წამყვანი გიტარა და რობერტ ტრუჰილო - ბას-გიტარა. ჯგუფმა გამოუშვა ცხრა სტუდიური ალბომი: Kill 'Em All (1983), Ride the Lightning (1984), Master of Puppets (1986), ...And Justice for All (1988), Metallica (1991), Load (1996), ReLoad (1997), St. Anger (2003) და Death Magnetic (2008). მეტალიკა ერთ-ერთი კომერციულად ყველაზე წარმატებული მძიმე მეტალ ჯგუფია, მხოლოდ აშშ-ში მათ გაყიდული აქვთ 57 მილიონზე მეტი ალბომი.მეტალიკას მიღებული აქვს ცხრა გრემის ჯილდო, მათგან ექვსი „საუკეთესო მეტალ შესრულებისათვის“ 1990, 1991, 1992, 1999, 2004 და 2009 წლებში. სიმღერა Enter Sandman განსაკუთრებით წარმატებული აღმოჩნდა ჯგუფისთვის: იგი წარდგენილი იქნა ნომინაციაზე „საუკეთესო როკ სიმღერა“ გრემის დაჯილდოების ცერემონიალზე და 1992 წელს მიიღო MTV-ის მუსიკალური ვიდეოების ჯილდო „საუკეთესო მეტალ ვიდეოსთვის“. ამ კატეგორიაში ჯგუფმა მეორე ჯილდოც მიიღო სიმღერისთვის Until it Sleeps. მეტალიკა ოთხჯერ იყო წარდგენილი ნომინაციაზე „ფავორიტი მძიმე მეტალ/მძიმე როკ შემსრულებელი“ ამერიკული მუსიკის დაჯილდოებაზე, რომელზეც ორჯერ გაიმარჯვა კიდეც, 1993 და 1997 წლებში. სამჯერ იქნა ნომინირებული მეტალიკა ბილბორდის მუსიკალურ დაჯილდოებაზე და სამჯერვე გაიმარჯვა. პირველად ჯგუფი ჯუნოს დაჯილდოებაზე წარდგენილი იქნა 1992 წელს სამ ნომინაციაში, მაგრამ არცერთში არ გაუმარჯვია; შემდეგ 2005 წელს მან ამავე დაჯილდოებაზე გაიმარჯვა ნომინაციაში „ჯეკ რიჩარდსონის წლის პროდიუსერი“ სიმღერით Some Kind of Monster. ჯამში მეტალიკას მიღებული აქვს 22 ჯილდო 57 ნომინაციიდან.


== ამერიკული მუსიკის დაჯილდოება ==
ამერიკული მუსიკის დაჯილდოება ამერიკის წლიური დაჯილდოების ცერემონიაა, დაარსებული დიკ კლარკის მიერ 1973 წელს. მეტალიკას მიღებული აქვს ორი ჯილდო ექვსი ნომინაციიდან.


== ბილბორდის მუსიკალური დაჯილდოება ==
ბილბორდის მუსიკალური დაჯილდოება ტარდება ყოველი წლის დეკემბერში და დასპონსორებულია ჟურნალ ბილბოლრდის მიერ. მეტალიკამ მიიღო სამი ჯილდო.


== გრემის დაჯილდოება ==
გრემის დაჯილდოების ცერემონიას ყოველწლიურად მართავს ჩანაწერების ხელოვნებისა და მეცნიერების ეროვნული აკადემია. მეტალიკას მიღებული აქვს ცხრა ჯილდო თვრამეტი ნომინაციიდან.


== ჯუნოს დაჯილდოება ==
ჯუნოს დაჯილდოება ტარდება ჩანაწერების ხელოვნებისა და მეცნიერების კანადის აკადემიის მიერ. მეტალიკამ მიიღო ერთი ჯილდო ხუთი ნომინაციიდან.


== Kerrang! დაჯილდოება ==
Kerrang! დაჯილდოება ყოველწლიური ცერემონიაა, რომელიც ტარდება ბრიტანული როკ ჟურნალ Kerrang!-ის მიერ. მეტალიკამ ხუთი ჯილდო მიიღო ამ ცერემონიაზე.


== მეტეორის მუსიკალური დაჯილდოება ==
მეტეორის მუსიკალური დაჯილდოება წარმოადგენს ირლანდიის ეროვნული დაჯილდოების ცერემონიალს, რომლის დისტრიბუციას ახორციელებს MCD Productions. მეტალიკამ მიიღო ერთი ნომინაცია.


== MTV Europe-ის მუსიკალური დაჯილდოება ==
MTV Europe-ის მუსიკალური დაჯილდოება MTV Europe-ის მიერ 1994 წელს დაარსებული ყოველწლიური დაჯილდოების ცერემონიაა. მეტალიკამ მიიღო ოთხი ნომინაცია.


== MTV-ის მუსიკალური ვიდეოების დაჯილდოება ==
MTV-ის მუსიკალური ვიდეოების დაჯილდოება MTV-ის მიერ 1984 წელს დაარსებული ყოველწლიური დაჯილდოების ცერემონიაა. მეტალიკამ მიიღო ორი ჯილდო თორმეტი ნომინაციიდან.


== Q-ის დაჯილდოება ==
Q-ის დაჯილდოება ტარდება ყოველწლიურად მუსიკალურ ჟურნალ Q-ის მიერ. მეტალიკამ მიიღო ერთი ნომინაცია.


== რადიოს მუსიკალური დაჯილდოება ==
რადიოს მუსიკალური დაჯილდოება ყოველწლიური დაჯილდოების შოუა, რომელიც ეძღვნება 40 საუკეთესო მეინსტრიმულ სიმღერას. მეტალიკამ მიიღო ორი ნომინაცია.


== მსოფლიო მუსიკალური დაჯილდოება ==
მსოფლიო მუსიკალური დაჯილდოება 1989 წელს დაარსებული ყოველწლიური დაჯილდოების შოუა, რომელიც ტარდება H.S.H.-ის პრინც ალბერტ II-ის პატრონაჟით. ჯონ მარტინოტი შოუს თანადამაარსებელი და აღმასრულებელი პროდიუსერია.
