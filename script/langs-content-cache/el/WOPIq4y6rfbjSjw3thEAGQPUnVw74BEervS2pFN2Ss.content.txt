Σεναριογράφος αποκαλείται αυτός που γράφει το σενάριο ενός τηλεοπτικού ή κινηματογραφικού ή θεατρικού έργου. Ο σεναριογράφος όχι μόνο γράφει την υπόθεση του σχετικού έργου, αλλά είναι και αυτός που επινοεί και τους χαρακτήρες του έργου.


== Ορολογία ==
Στη εποχή του βωβού κινηματογράφου, οι συγγραφείς που θεωρούνται πλέον σεναριογράφοι υποδηλώνονταν με διάφορους όρους όπως φωτογράφος, συγγραφέας φωτογραφιών, δραματουργός φωτογραφιών και θεατρικός σεναριογράφος.  Ο ιστορικός σεναριογράφου Στίβεν Μάρας σημειώνει ότι αυτοί οι πρώτοι συγγραφείς θεωρούνταν συχνά ως συγγραφείς των ταινιών όπως παρουσιάζονται και υποστηρίζει ότι δεν μπορούν να εξισωθούν με ακρίβεια με τους σημερινούς σεναριογράφους επειδή ήταν υπεύθυνοι για ένα τεχνικό προϊόν, ένα σύντομο «σενάριο», ή «σύνοψη» που είναι μια γραπτή σύνοψη του τι πρόκειται να κινηματογραφηθεί.


== Επάγγελμα ==
Η σεναριογραφία είναι ελεύθερο επάγγελμα. Δεν απαιτείται εκπαίδευση για να είσαι επαγγελματίας σεναριογράφος, μόνο καλές ικανότητες αφήγησης και φαντασία. Οι σεναριογράφοι δεν είναι μισθωτοί υπάλληλοι αλλά συμβασιούχοι ελεύθεροι επαγγελματίες. Αυτό που διαχωρίζει έναν επαγγελματία σεναριογράφο από έναν ερασιτέχνη σεναριογράφο είναι ότι οι επαγγελματίες σεναριογράφοι συνήθως εκπροσωπούνται από ένα γραφείο ταλέντων. Επίσης, οι επαγγελματίες σεναριογράφοι συχνά δεν εργάζονται δωρεάν, αλλά οι ερασιτέχνες σεναριογράφοι συχνά εργάζονται χωρίς πληρωμή και θεωρούνται «συγγραφείς σε εκπαίδευση». Τα αυτόκλητα σενάρια γράφονται συνήθως από άγνωστους επαγγελματίες ή ερασιτέχνες σεναριογράφους.
Υπάρχει μια πληθώρα επίδοξων σεναριογράφων που προσπαθούν να εισέλθουν στην κινηματογραφική βιομηχανία, αλλά συχνά χρειάζονται χρόνια δοκιμής και λάθους, αποτυχίας και σκληρής επιμονής για να επιτευχθεί η επιτυχία. 


== Κινηματογραφική βιομηχανία ==
Κάθε σενάριο ξεκινά με μια σκέψη ή ιδέα, και οι σεναριογράφοι χρησιμοποιούν τις ιδέες τους για να γράψουν σενάρια, με σκοπό να τα πουλήσουν και να παραχθούν. Σε ορισμένες περιπτώσεις, το σενάριο βασίζεται σε ένα βιβλίο ή μια ιστορία ζωής ενός ατόμου, η οποία έχει προσαρμοστεί από τον σεναριογράφο. Τις περισσότερες φορές, ένα κινηματογραφικό έργο ξεκινά από έναν σεναριογράφο. Ο εμπνευστής του έργου αναλαμβάνει την αποκλειστική γραπτή εργασία.Οι σεναριογράφοι που παρουσιάζουν συχνά νέα έργα, πρωτότυπα ή προσαρμοσμένα, συχνά δεν χρειάζεται να ανησυχούν για τον ανταγωνισμό για εργασίες και συχνά είναι πιο επιτυχημένοι. Όταν γίνεται λόγος για ένα έργο που θέλει να γίνει ένα κινηματογραφικό στούντιο, μια εταιρεία παραγωγής ή ένας παραγωγός, αναφέρονται ως «ανοιχτές» αναθέσεις. Οι ανοιχτές εργασίες είναι πιο ανταγωνιστικές. Εάν οι σεναριογράφοι ανταγωνίζονται για μια ανοιχτή εργασία, οι πιο καθιερωμένοι σεναριογράφοι συνήθως κερδίζουν τις αναθέσεις. Ένας σεναριογράφος μπορεί επίσης να προσεγγιστεί και να του προσφερθεί προσωπικά μια γραπτή εργασία.
