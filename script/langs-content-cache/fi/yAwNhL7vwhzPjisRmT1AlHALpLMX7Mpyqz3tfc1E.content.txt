Ip Man eli Yip Man (kiin.: 葉問; Jyutping: Jip6 Man6; 1893 – 2. joulukuuta 1972) oli kiinalainen Wing Tsungin opettaja. Yksi hänen tunnetuimmista oppilaistaan oli Bruce Lee. Ip Mania pidetään nykyaikaisen Wing Tsungin suurmestarina.


== Elämäkerta ==
Ip syntyi Ip Oi-dor’lle ja Ng Shuille nelilapsiseen perheeseen. Hän kasvoi varakkaan perheensä kanssa Foshanissa, Guangdongissa ja sai perinteisen kiinalaisen koulutuksen.
Ip alkoi harjoitella Wing Tsungia 13-vuotiaana Chan Wah-shunin opastuksella. Ip oli samalla Chanin viimeinen oppilas. Chan oli tuolloin 70-vuotias eikä voinut enää ikänsä puolesta opettaa. Chanin kuoltua kolme vuotta myöhemmin hänen toiseksi vanhin oppilaansa Ng Chung-sok opetti Ipille suurimman osan pojan taidoista ja tekniikoista.
15-vuotiaana Ip muutti Hongkongiin sukulaisensa Leung Fut-tingin avustuksella. Vuotta myöhemmin Ip opiskeli St. Stephen’s Collegessa, joka oli tarkoitettu rikkaille perheille ja ulkomaalaisille, jotka asuivat Hongkongissa. Opiskelunsa aikana hän näki ulkomaalaisen poliisin hakkaavan naista ja puuttui väliin. Poliisi oli aikeissa hyökätä Ipin kimppuun, mutta hän onnistui lyömään poliisin maihin ja pakeni kouluunsa. Myöhemmin eräs Ipin luokkalainen kertoi tapahtuneesta vanhemmalle miehelle, joka asui samassa kerrostalossa. Mies tapasi Ipin ja kysyi, mitä kamppailua hän harrasti. Ip kertoi, ettei hän ole taidoiltaan kovin ”hyvä”. Mies haastoi Ipin tappeluun ja voitti myös pojan. Mies kertoi Ipille olevan Leung Bik, Chanin opettajan Leung Janin poika. Tappelun jälkeen Ip jatkoi harjoitteluaan Leung Bikin oppilaana.
Myöhemmin 24-vuotiaana Ip palasi takaisin Foshaniin, jossa hän ryhtyi poliisiksi. Samalla hän ryhtyi opettamaan Wing Tsungia työkavereilleen, ystävilleen ja sukulaisilleen, mutta ei ylläpitänyt virallista Wing Tsung -koulua. Toisen Kiinan–Japanin sodan syttyessä Ip muutti oppilaansa Kwok Fun luokse sodan ajaksi. Sodan loputtua vuonna 1945 Ip palasi takaisin Foshaniin, jossa hän jatkoi poliisin ammattiaan.
Vuonna 1949 Ip muutti Hongkongiin, kun Kiinan kommunistinen puolue perusti nykyisen Kiinan Manner-Kiinan alueelle 1. lokakuuta 1949. Ipin tiedetään olleen riippuvainen oopiumista ja muutettuaan Hongkongiin hän sai ostettua sitä pimeiltä markkinoilta. Oopiumi oli erittäin kallista tuolloin ja Ipin täytyi saada tasaisesti tuloja voidakseen polttaa oopiumia ja elättää Foshanissa asuvaa perhettään. Hän päätti avata Wing Tsung -koulun ja opettaa ihmisiä. Liiketoiminta oli aluksi heikkoa, kun oppilaat tyypillisesti harjoittelivat vain pari kuukautta. Ip muutti koulunsa tiloja kaksi kertaa Hongkongissa, aluksi Sham Shui Pon ja myöhemmin Yau Ma Tein alueelle. Osa hänen oppilaistaan oli tullut päteviksi opettamaan Wing Tsungia omissa harjoitussaleissaan. Myöhemmin vuonna 1967 Ip ja muutamat hänen oppilaansa perustivat Hongkong Wing Tsung -urheiluyhdistyksen.
Ip Man kuoli 2. joulukuuta 1972 keuhkosyöpään ollessaan 79-vuotias.
Kuolemansa jälkeen Foshaniin perustettiin hänen kunniaksi museo Ip Man Tong.


== Ip Man populaarikulttuurissa ==
Ip Man alkoi esiintyi televisiossa 2000-luvulla, kun The Legend of Bruce Lee -televisiosarjassa häntä näytteli Yu Cheng Hui.
Sitä ennen vuonna 1998 Ip Manista oli jo suunnitteilla elokuva, mutta varsinainen toteutus tapahtui kymmenen vuotta myöhemmin. Joulukuussa 2008 sai ensi-iltansa Ip Man -niminen elokuva, joka perustui löyhästi Ip Manin elämään toisen Kiinan–Japanin sodan aikana. Myös Ipin vanhin poika Ip Chun esitettiin elokuvassa. Kaksi vuotta myöhemmin elokuvasta tuotettiin jatko-osa Ip Man 2, joka keskittyi Ip Manin elämään Hongkongissa. Vuonna 2015 ilmestyi kolmas elokuva Ip Man 3. Wing Tsung -mestaria esitti Donnie Yen.
Kesällä 2010 Ip Manista esitettiin erillinen elokuva The Legend is Born – Ip Man, jossa pääosaa esittää Dennis To. Elokuva keskittyi Ip Manin varhaiseen elämään, kun hän opettelee Wing Tsungia.
